# Booking System

Detta är ett eget projekt där jag utvecklar ett bokningssystem med fokus på backendlogik, databashantering och systemstruktur.

## Tekniker
- .NET
- SQL
- Next.js

## Funktioner
- Hantering av bokningar
- Databasstruktur för lagring av data
- Backendlogik för att hantera systemflöden

## Syfte
Projektet är utvecklat för att få praktisk erfarenhet av att bygga ett system från grunden och arbeta med både frontend och backend.
